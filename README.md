# Make Language

Make Language is a power full tool and framework that allow you create your own programming language and has a great api that you can add more functionality.

## Features

- var
- string
- number
- const
- if
- elif
- else
- for
- while
- array
- functions
- class
- math operations

Any of these functionality can be modify using a simple conf.json file

## API Reference

#### Var

```
  var a = 1
```

#### Const

```
  const a = 1
  a = 2
```

#### String

```
  var a = "hello world"
  print(a.replace("e","u"))
  print(a +  " 123")
  print(a * 2)
```

#### Number

```
  var a = 25
  print(a + 5) // 30
  print(a - 5) // 25
  print(a * 2) // 50
  print(a / 5) // 5
  print(a ~ 2) // 5
  print(100 ~ 2) // 10
```

#### Class

```
  class Test {
    func a(){
      this.b("Hello ")
    }

    func b(hello){
      print(hello + "World")
    }
  }
```
